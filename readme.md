# Sorter 项目

这是一个排序算法实现项目，包含多种排序算法的实现。

## 项目结构
```
.gitignore
src/
  __init__.py
  __pycache__/
    __init__.cpython-312.pyc
test/
  test.py
```

## 安装
运行 `python setup.py install` 或 `pip install sorter` 安装项目。

## 使用方法
导入 `src` 模块中的 `Sorter` 类，调用相应的排序方法。

## 测试
运行 `test/test.py` 中的单元测试。
